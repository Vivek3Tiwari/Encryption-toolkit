Encryption Toolkit

Explore multiple encryption techniques in one polished interface.

Security: Unknown

Caesar Cipher

Shift each alphabetic character by a fixed number.

Example: HELLO with shift 3 -> KHOOR

Vigenere Cipher

Use a keyword to apply a variable shift per character.

Example: ATTACKATDAWN + LEMON -> LXFOPVEFRNHR

AES-GCM

Strong symmetric encryption using your passphrase (Web Crypto API).

Use 12+ character passphrases for better security.

DES

Encrypt/decrypt text using DES (CryptoJS).

3DES is safer than DES but still legacy.

SHA-256 Hash

Generate a secure one-way digest for integrity checks.

Hill Cipher (2x2)

Matrix-based substitution cipher using a 2x2 key matrix.

Example key 3,3,2,5 and text HELP -> HIAT

Playfair Cipher

Digraph substitution using a 5x5 key square (I/J combined).

Pairs repeated letters with filler X (I/J treated same).

Columnar Transposition

Reorder characters by reading columns based on a keyword.

Keyword controls column read order alphabetically.

Rail Fence Cipher

Zig-zag transposition cipher based on rail count.

Rail visualization appears after encrypt/decrypt.
